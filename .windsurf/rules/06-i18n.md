# 国际化

## 支持语言

| 语言 | 代码 | 状态 |
|------|------|------|
| 简体中文 | zh_CN | ✅ 默认 |
| English | en_US | ✅ 支持 |

## 实现方式

### QML 翻译

```qml
import QtQuick

Text {
    text: qsTr("Welcome to EnhanceVision")
    text: qsTr("Processing %1 of %2 files").arg(current).arg(total)
    text: qsTr("%n file(s) selected", "", count)
}
```

### C++ 翻译

```cpp
#include <QCoreApplication>

QString message = tr("Processing completed");
QString error = tr("Failed to load file: %1").arg(fileName);
```

### 翻译上下文

```qml
Text {
    text: qsTranslate("FileList", "Open")
}

Text {
    text: qsTranslate("Settings", "Open")
}
```

## 翻译文件管理

### 文件位置

```
resources/i18n/
├── app_zh_CN.ts
├── app_en_US.ts
├── app_zh_CN.qm
└── app_en_US.qm
```

### 翻译文件格式

```xml
<!DOCTYPE TS>
<TS version="2.1" language="zh_CN">
<context>
    <name>MainPage</name>
    <message>
        <source>Welcome to EnhanceVision</source>
        <translation>欢迎使用 EnhanceVision</translation>
    </message>
</context>
</TS>
```

## 构建集成

### CMake 配置

```cmake
find_package(Qt6 REQUIRED COMPONENTS LinguistTools)

set(TS_FILES
    "${CMAKE_SOURCE_DIR}/resources/i18n/app_zh_CN.ts"
    "${CMAKE_SOURCE_DIR}/resources/i18n/app_en_US.ts"
)

qt_add_translations(EnhanceVision
    TS_FILES ${TS_FILES}
    LUPDATE_OPTIONS -no-obsolete
    LRELEASE_OPTIONS -compress
)
```

### 翻译更新命令

```powershell
cmake --build . --target EnhanceVision_lupdate
cmake --build . --target EnhanceVision_lrelease
```

## 运行时语言切换

### C++ 语言管理器

```cpp
class LanguageController : public QObject {
    Q_OBJECT
    Q_PROPERTY(QString currentLanguage READ currentLanguage NOTIFY languageChanged)
    Q_PROPERTY(QStringList availableLanguages READ availableLanguages CONSTANT)
    
public:
    Q_INVOKABLE void setLanguage(const QString& languageCode);
    
signals:
    void languageChanged();
    
private:
    QTranslator m_translator;
    QString m_currentLanguage;
};
```

### QML 语言切换

```qml
ComboBox {
    model: LanguageController.availableLanguages
    onCurrentIndexChanged: {
        LanguageController.setLanguage(model[currentIndex])
    }
}
```

## 持久化

```cpp
void LanguageController::setLanguage(const QString& languageCode) {
    QSettings settings;
    settings.setValue("language", languageCode);
}
```

## 规则

### 必须翻译
- 所有用户可见的 UI 文本
- 错误消息和提示
- 对话框标题和按钮文本

### 不翻译
- 日志消息
- 配置键名
- API 路径
- 文件名和路径
