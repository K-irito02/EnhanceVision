# 国际化

## 语言范围

| 语言 | 代码 | 文件 |
|------|------|------|
| 默认 | `zh_CN` | `app_zh_CN.ts` |
| 支持 | `en_US` | `app_en_US.ts` |

## 文本规则

| 位置 | 方法 |
|------|------|
| QML | `qsTr()` |
| C++ | `tr()` |

1. **禁止硬编码用户可见文本**
2. **动态文本使用 `%N` 占位**：`qsTr("处理 %1 个文件").arg(count)`

## 更新流程

```powershell
# 1. 更新翻译源文件
cmake --build . --target EnhanceVision_lupdate

# 2. 编辑翻译（Qt Linguist）
linguist resources/i18n/app_zh_CN.ts

# 3. 生成二进制文件
cmake --build . --target EnhanceVision_lrelease
```

## UI 适配约束

1. **按钮最小宽度 80px**：容纳中英文本
2. **文本容器高度自适应**：禁止固定高度
3. **长文本设置 `elide`**：`elide: Text.ElideRight`
4. **功能完成后切换英文验证布局**

## UI Adaptation Constraints

1. **Button minimum width 80px**: Accommodate Chinese and English text
2. **Text container height adaptive**: Prohibit fixed height
3. **Long text set `elide`**: `elide: Text.ElideRight`
4. **Switch to English to verify layout after function completion**

## Tooltip Internationalization

1. **All tooltips must use `qsTr()`**: Support language switching
2. **Tooltip naming convention**: Use descriptive text (e.g., "Click to view source")
3. **Dynamic tooltips**: Update based on current state
4. **Tooltip length**: Keep concise, avoid overly long descriptions

## Translation File Management

1. **Run `lupdate` after adding new translatable strings**
2. **Remove obsolete translations**: Clean up unused entries
3. **Maintain translation consistency**: Use same terminology across files
4. **Test language switching**: Verify all UI text updates correctly
