# UI 设计规范

## 主题系统

- 颜色、间距、圆角、动效统一由 Theme 管理
- 亮/暗主题均需完整适配
- 禁止组件内硬编码主题色

## 排版与层级

- 字体族、字号、字重使用 Typography 统一定义
- 标题、正文、说明层级清晰，不混用
- 文案对齐方式与间距保持一致

## 交互状态

- 组件具备默认/悬停/按下/禁用/焦点状态
- 关键操作提供明确反馈（成功/失败/进行中）
- 空状态、异常状态必须有可读提示

## 布局与响应式

- 常见窗口尺寸下不遮挡、不溢出、可点击
- 区块间距使用统一 spacing token
- 固定区与滚动区边界清晰

## 可访问性

- 键盘可达，焦点可见
- 文本与背景具备可读对比度

## 本文件边界

- 仅定义 UI 设计系统
- 国际化适配见 `06-i18n.mdc`
- 性能相关见 `07-performance.mdc`
