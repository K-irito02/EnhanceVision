# UI 设计规范

## 主题系统

- 颜色、间距、圆角、动效统一由 `Theme.qml` 管理
- 亮/暗主题均需完整适配
- ❌ 禁止组件内硬编码主题色

## 样式文件

| 文件 | 职责 |
|------|------|
| `Theme.qml` | 主题单例，颜色/间距/圆角 |
| `Colors.qml` | 颜色定义 |
| `Typography.qml` | 字体定义 |

## 排版与层级

- 字体族、字号、字重使用 `Typography.qml` 统一定义
- 标题、正文、说明层级清晰，不混用
- 文案对齐方式与间距保持一致

## 交互状态

| 状态 | 说明 |
|------|------|
| 默认 | 正常显示 |
| 悬停 | 鼠标悬停 |
| 按下 | 鼠标按下 |
| 禁用 | 不可用 |
| 焦点 | 键盘焦点 |

- 关键操作提供明确反馈（成功/失败/进行中）
- 空状态、异常状态必须有可读提示

## 布局与响应式

- ✅ 常见窗口尺寸下不遮挡、不溢出、可点击
- ✅ 区块间距使用统一 spacing token
- ✅ 固定区与滚动区边界清晰

## 可访问性

- ✅ 键盘可达，焦点可见
- ✅ 文本与背景具备可读对比度
