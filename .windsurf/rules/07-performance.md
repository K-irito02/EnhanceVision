# 性能优化

## 线程模型

| 线程 | 职责 |
|------|------|
| 主线程 | UI 渲染与轻量逻辑 |
| 工作线程 | IO、图像处理、AI 推理、视频处理 |

1. **跨线程通信使用信号槽（队列连接）**
2. **推理前在主线程快照模型与参数**
3. **工作线程只持快照，不访问主线程对象**

## 并发调度

| 配置项 | 值 |
|--------|-----|
| AI 引擎池 | 1 个引擎（线性任务） |
| 任务超时 | 5 分钟 |
| 重试策略 | 指数退避，最大 3 次 |

## GPU 内存管理

1. **推理前检查 GPU 可用内存**
2. **OOM 时自动重试更小分块**：128→96→64
3. **模型切换前清理 Vulkan allocator**
4. **引擎释放前同步 Vulkan 队列**

## QML 渲染

1. **列表优先 `ListView`**：避免大规模 `Repeater`
2. **重组件使用 `Loader`**：延迟加载
3. **图像设置 `sourceSize`**：最大 1920×1080
4. **进度更新节流**：发射间隔 ≥ 100ms

## 动画规范

1. **优先动画 `opacity/transform`**
2. **避免高频动画 `width/height/x/y`**
3. **控制同屏持续动画数量**

## 性能基线

| 指标 | 要求 |
|------|------|
| 关键交互 | 无明显卡顿 |
| 列表滚动 | 保持流畅 |
| AI 推理 OOM | 自动恢复，不崩溃 |
