# EnhanceVision 开发规则与约束

## 一、线程安全

1. **禁止在工作线程直接操作 UI**：信号槽连接必须使用 `Qt::QueuedConnection`
2. **模型加载必须阻塞检查**：`loadModel()` 前检查 `m_isProcessing`，拒绝加载或等待完成
3. **共享数据访问必须加锁**：`QMutexLocker` 保护 `m_taskQueue`、`m_processingTasks` 等队列
4. **GPU 同步使用事件而非 sleep**：使用 `VkCompute::submit_and_wait()` + fence，禁止 `QThread::msleep()` 轮询
5. **进度信号节流**：发射间隔 ≥ 100ms，使用时间戳判断

## 二、内存管理

1. **GPU 分配器及时清理**：模型卸载/推理结束时调用 `clear()`，置于 `QMutexLocker` 内
2. **大图像分块处理**：超过 2K 分辨率必须分块，单块不超过 512MB
3. **模型切换先卸载**：`loadModel()` 前必须调用 `unloadModel()`，禁止同时持有多个模型
4. **视频帧处理限流**：同时缓存的解码帧数 ≤ 5 帧

## 三、资源释放

1. **信号连接使用 QMetaObject::Connection**：存储连接句柄，切换引擎时调用 `disconnect()`
2. **处理器使用 QSharedPointer**：`m_activeVideoProcessors` 等容器使用智能指针自动管理
3. **临时文件必须清理**：任务完成/取消/失败时删除输出目录中的临时文件
4. **Shader 组件显式销毁**：`ShaderEffect` 的 `source` 置空后再销毁父组件

## 四、UI 性能

1. **耗时操作必须异步**：模型加载、文件读取、图像处理使用 `QtConcurrent::run()`
2. **ListView 使用 delegate 复用**：设置 `cacheBuffer`，禁止在 delegate 内创建复杂组件
3. **预览图限制尺寸**：`sourceSize` 最大 1920×1080，禁止加载原图预览
4. **进度更新批量合并**：使用定时器批量更新，禁止每帧触发 UI 重绘

## 五、响应式布局

1. **禁止固定宽度**：面板使用 `Layout.preferredWidth` + `Layout.minimumWidth`
2. **设置最小窗口**：`minimumWidth: 800`、`minimumHeight: 600`
3. **字体使用相对单位**：`font.pixelSize: Theme.typography.body`，禁止硬编码像素值
4. **折叠状态必须重算布局**：折叠时触发 `Layout.requestGeometry()`

## 六、国际化

1. **所有用户可见文本必须包裹**：QML 用 `qsTr()`，C++ 用 `tr()`
2. **按钮最小宽度 80px**：容纳中英文本，设置 `Layout.minimumWidth`
3. **翻译文件同步更新**：代码提交前运行 `lupdate` + `lrelease`
4. **动态文本使用 %N 占位**：`qsTr("处理 %1 个文件").arg(count)`，禁止字符串拼接

## 七、主题系统

1. **禁止硬编码颜色**：统一使用 `Theme.colors.primary` 等语义变量
2. **图标使用 ColoredIcon**：禁止直接使用 `Image`，必须支持动态着色
3. **主题切换使用 Behavior**：颜色属性添加 `Behavior on xxx { ColorAnimation { duration: 150 } }`
4. **新增组件必须测试双主题**：亮色/暗色切换后检查对比度

## 八、翻译切换适配

1. **文本容器高度自适应**：禁止固定高度，使用 `contentHeight` 或 `implicitHeight`
2. **长文本设置 elide**：`elide: Text.ElideRight` + `maximumLineCount`
3. **按钮使用 implicitWidth**：禁止固定宽度，允许文本撑开
4. **测试英文界面**：功能完成后必须切换英文验证布局

## 九、其他约束

1. **会话切换必须停止任务**：切换前调用 `cancelAllTasks()` 并等待完成
2. **批量导入限制数量**：单次导入 ≤ 100 个文件，超出分批处理
3. **取消操作必须清理资源**：`cancel()` 后检查并释放处理器、临时文件、内存
4. **异步操作必须处理失败**：`onError` 回调中恢复 UI 状态，禁止残留 loading 态
5. **日志禁止敏感路径**：使用 `[FILE]` 占位或相对路径，禁止输出完整用户路径
