# 01 项目基线与边界规则

## 技术基线

- Qt：`6.10.2`
- 编译器：`MSVC 2022`
- C++：`C++20`
- 构建系统：`CMake Presets`

## 目录边界

- 后端：`src/` + `include/`
- 前端：`qml/`
- 资源：`resources/`
- 文档：`docs/`
- 测试：`tests/`

## 开发边界

- UI 展示与交互放 QML
- 业务、IO、处理逻辑放 C++
- 公共资源统一进 `resources/`，禁止散落引用

## 变更边界

- 新功能必须说明影响范围（前端/后端/资源/构建）
- 非需求驱动不做大规模重构
- 任何“临时方案”必须可回收并有后续清理计划
