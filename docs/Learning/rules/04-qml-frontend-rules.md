# 04 QML 前端规范

## 文件与命名

- QML 文件名使用 PascalCase
- `id`、属性、信号使用 camelCase
- 私有状态使用 `_` 前缀

## 组件化

- 页面放 `qml/pages/`
- 可复用块放 `qml/components/`
- 基础控件放 `qml/controls/`
- 样式集中在 `qml/styles/`

## 绑定与交互

- 优先声明式绑定，避免命令式同步
- 复杂逻辑抽函数或下沉 C++
- 用户可见字符串必须 `qsTr()`

## 禁止项

- 在绑定表达式中写复杂计算
- 在委托中堆叠重量级对象
- 复制粘贴式重复组件实现
