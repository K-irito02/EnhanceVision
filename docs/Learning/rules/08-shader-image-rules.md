# 08 Shader 与图像处理规范

## Shader 组织

- Shader 源码统一放 `resources/shaders/`
- QML 封装统一放 `qml/shaders/`
- 命名与效果一一对应，避免同义重复文件

## 参数一致性

- 参数范围、默认值、步长在 UI 与 Shader 中保持一致
- 禁止使用会吞掉 `0` 的默认值写法
- 预览与导出参数必须同源

## 质量与性能

- 优先单次采样复用，减少重复计算
- 分支逻辑尽量用数学插值替代
- 新增效果必须做视觉回归对比

## 稳定性

- Shader 路径可追踪、可定位
- 编译失败有降级或错误提示
