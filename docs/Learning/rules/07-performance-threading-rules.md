# 07 性能与多线程规范

## 线程规则

- 主线程只处理 UI 与轻量逻辑
- IO、图像处理、推理、视频处理必须后台执行
- 跨线程通信使用信号槽（队列连接）

## QML 性能

- 列表优先 `ListView`，避免大规模 `Repeater`
- 重组件使用 `Loader` 延迟加载
- 图像明确 `sourceSize` 与异步加载策略

## 动画与渲染

- 优先动画 `opacity/transform`
- 减少触发布局重算的属性动画
- 同屏持续动画数量受控

## 性能门槛

- 关键交互无明显卡顿
- 滚动与预览保持流畅
- 新功能不得明显拉低现有性能基线
