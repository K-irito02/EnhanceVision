## 五、三图一致性要求

### 统一的设计 DNA

| 维度 | 统一标准 |
|------|----------|
| **品牌色** | 三张图均以 `#002FA7` 克莱因蓝为主色 |
| **辅助色** | 统一使用 `#1A56DB` / `#3B82F6` 渐变系 |
| **眼睛符号** | welcome 和 header 中的眼睛 = app.ico 的同一设计语言 |
| **光效风格** | 统一的泛光(bloom)、粒子(particle)、高光(specular)处理方式 |
| **立体感** | 统一的微 3D / 微浮雕质感 |
| **背景基调** | 统一深色系（`#050A18` ~ `#0F1D3A` 范围） |
| **文字字体** | welcome 和 header 使用相同的字体族 |

### 推荐生成顺序

```
1. 先生成 app.ico（256×256）
   └─ 确定眼睛的核心设计语言和配色方案
   
2. 再生成 welcome.bmp（500×314）
   └─ 将确定的眼睛设计融入大场景，添加背景和文字
   
3. 最后生成 header.bmp（150×57）
   └─ 提取眼睛迷你版 + 品牌名文字，紧凑排版
```

## 六、生成后的后续步骤

1. **格式转换**
   - welcome.bmp / header.bmp：直接保存为 24位或 32位 BMP
   - app.ico：先用 PNG 导出 256×256，再用工具（如 ImageMagick / ICO Convertor）生成多尺寸 ICO

2. **放入项目路径**
   ```
   resources/
   ├── icons/
   │   ├── app_icon.ico          ← 替换现有
   │   └── app_icon.png          ← 同步更新 256 版本
   └── images/                    （如需要新建此目录）
       ├── welcome.bmp            ← 新增
       └── header.bmp             ← 新增
   ```

3. **QML 集成验证**
