# 国际化

## 翻译文件管理

### 文件位置

```
resources/i18n/
├── app_zh_CN.ts
├── app_en_US.ts
├── app_zh_CN.qm
└── app_en_US.qm
```

### 翻译文件格式

```xml
<!DOCTYPE TS>
<TS version="2.1" language="zh_CN">
<context>
    <name>MainPage</name>
    <message>
        <source>Welcome to EnhanceVision</source>
        <translation>欢迎使用 EnhanceVision</translation>
    </message>
</context>
</TS>
```

## 构建集成

### CMake 配置

```cmake
find_package(Qt6 REQUIRED COMPONENTS LinguistTools)

set(TS_FILES
    "${CMAKE_SOURCE_DIR}/resources/i18n/app_zh_CN.ts"
    "${CMAKE_SOURCE_DIR}/resources/i18n/app_en_US.ts"
)

qt_add_translations(EnhanceVision
    TS_FILES ${TS_FILES}
    LUPDATE_OPTIONS -no-obsolete
    LRELEASE_OPTIONS -compress
)
```

### 翻译更新命令

```powershell
cmake --build . --target EnhanceVision_lupdate
cmake --build . --target EnhanceVision_lrelease
```

## 运行时语言切换

### C++ 翻译器管理 (Application 类)

```cpp
// Application.h
class Application : public QObject {
    Q_OBJECT
public:
    void setupTranslator();
    bool switchTranslator(const QString& language);

private slots:
    void onLanguageChanged();

private:
    QSharedPointer<QTranslator> m_translator;
    QSharedPointer<QTranslator> m_qtTranslator;
};

// Application.cpp
void Application::setupTranslator()
{
    QString language = SettingsController::instance()->language();
    switchTranslator(language);

    connect(SettingsController::instance(), &SettingsController::languageChanged,
            this, &Application::onLanguageChanged);
}

bool Application::switchTranslator(const QString& language)
{
    if (m_translator) {
        QCoreApplication::removeTranslator(m_translator.data());
    }
    if (m_qtTranslator) {
        QCoreApplication::removeTranslator(m_qtTranslator.data());
    }

    m_translator.reset(new QTranslator);
    QString qmFile = QString(":/i18n/app_%1.qm").arg(language);
    if (m_translator->load(qmFile)) {
        QCoreApplication::installTranslator(m_translator.data());
    }

    m_qtTranslator.reset(new QTranslator);
    QString qtQmFile = QString("qtbase_%1").arg(language);
    QString qtTranslationsPath = QLibraryInfo::path(QLibraryInfo::TranslationsPath);
    if (m_qtTranslator->load(qtQmFile, qtTranslationsPath)) {
        QCoreApplication::installTranslator(m_qtTranslator.data());
    }

    return true;
}

void Application::onLanguageChanged()
{
    QString language = SettingsController::instance()->language();
    switchTranslator(language);

    if (m_mainWidget && m_mainWidget->engine()) {
        m_mainWidget->engine()->retranslate();
    }
}
```

### QML 语言切换 (Theme.qml)

```qml
pragma Singleton
import QtQuick
import EnhanceVision.Controllers

QtObject {
    property string language: "zh_CN"

    function toggleLanguage() {
        var newLang = (language === "zh_CN") ? "en_US" : "zh_CN"
        language = newLang
        SettingsController.language = newLang
    }

    function setLanguage(lang) {
        language = lang
        SettingsController.language = lang
    }

    Component.onCompleted: {
        language = SettingsController.language
    }
}
```

## 持久化

```cpp
void LanguageController::setLanguage(const QString& languageCode) {
    QSettings settings;
    settings.setValue("language", languageCode);
}
```

## 规则

### 必须翻译
- 所有用户可见的 UI 文本
- 错误消息和提示
- 对话框标题和按钮文本

### 不翻译
- 日志消息
- 配置键名
- API 路径
- 文件名和路径
