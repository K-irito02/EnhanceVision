# Shader模式参数修复与优化计划

## 问题诊断

### 问题1：降噪参数不起作用

**根本原因分析**：

1. **Shader执行顺序问题**：在 `fullshader.frag` 中，处理顺序为：
   - 降噪 (第12步) → 模糊 (第13步) → 锐化 (第14步)
   - 锐化（Unsharp Mask）会重新引入噪点，抵消降噪效果

2. **混合系数过小**：
   ```glsl
   rgb = mix(rgb, medianColor, denoise * 0.5);
   ```
   系数 `0.5` 使得降噪效果减半，用户设置 0.5 的降噪值实际只有 0.25 的效果

3. **阈值过高**：
   ```glsl
   if (denoise > 0.01)
   ```
   当用户设置较小值时（如 0.01-0.02），效果可能被跳过

4. **中值滤波实现效率低**：当前使用简单的冒泡排序找中值，效率较低

### 问题2：参数独立性验证

经过代码审查，所有参数在 Shader 中是独立实现的，不存在相互阻塞的问题。但存在执行顺序影响：
- 降噪 → 模糊 → 锐化 的顺序会导致锐化重新引入噪点
- 建议调整顺序为：模糊 → 降噪 → 锐化

### 问题3：参数范围配置问题

| 参数 | 当前范围 | 问题 |
|------|----------|------|
| 亮度 | -1.0 ~ 1.0 | 范围过窄，专业软件通常 -100 ~ 100 |
| 对比度 | 0.0 ~ 2.0 | 范围合理，但UI显示应更直观 |
| 饱和度 | 0.0 ~ 2.0 | 范围合理 |

## 实施计划

### 第一阶段：修复降噪参数

#### 1.1 修改 Shader 执行顺序
**文件**: `resources/shaders/fullshader.frag`

将处理顺序调整为：
1. 曝光 → 亮度 → 对比度 → 饱和度 → 色相 → 伽马 → 色温 → 色调
2. 高光 → 阴影 → 晕影
3. **模糊 → 降噪 → 锐化**（调整顺序）

#### 1.2 增强降噪效果
**文件**: `resources/shaders/fullshader.frag`

- 移除阈值判断或降低阈值到 0.001
- 增加混合系数：`denoise * 0.5` → `denoise * 0.8`
- 优化中值滤波算法，使用更高效的排序方法

#### 1.3 优化锐化算法
**文件**: `resources/shaders/fullshader.frag`

- 锐化应在降噪后执行
- 添加边缘保护，避免锐化重新引入噪点

### 第三阶段：Shader 效果优化

#### 3.1 优化 fullshader.frag 中的效果系数

```glsl
// 降噪：增强效果
rgb = mix(rgb, medianColor, denoise * 0.8);  // 从 0.5 增加到 0.8

// 色温：增强效果
rgb.r = clamp(rgb.r + temperature * 0.2, 0.0, 1.0);  // 从 0.1 增加到 0.2
rgb.b = clamp(rgb.b - temperature * 0.2, 0.0, 1.0);

// 色调：增强效果
rgb.g = clamp(rgb.g + tint * 0.2, 0.0, 1.0);  // 从 0.1 增加到 0.2

// 高光/阴影：增强效果
float adjustment = highlights * factor * 0.3;  // 从 0.2 增加到 0.3
float adjustment = shadows * factor * 0.3;     // 从 0.2 增加到 0.3
```

#### 3.2 添加边缘保护的锐化算法

```glsl
// 锐化（带边缘保护）
if (sharpness > 0.001) {
    // 计算局部方差作为边缘检测
    float variance = ...;
    // 边缘区域减少锐化强度，避免放大噪点
    float edgeFactor = 1.0 - smoothstep(0.0, 0.1, variance);
    rgb = clamp(rgb + sharpness * sharpenAmount * edgeFactor, 0.0, 1.0);
}
```

## 技术细节

### 降噪算法优化方案

当前实现使用 3x3 中值滤波，存在的问题：
1. 冒泡排序效率低
2. 混合系数过小
3. 与锐化顺序冲突

优化方案：
1. 使用选择排序找中值（只需找到第5个元素）
2. 增加混合系数到 0.8
3. 调整执行顺序：模糊 → 降噪 → 锐化

### 参数效果系数对照表

| 参数 | 当前系数 | 建议系数 | 原因 |
|------|----------|----------|------|
| 降噪混合 | 0.5 | 0.8 | 效果太弱 |
| 色温偏移 | 0.1 | 0.2 | 效果不明显 |
| 色调偏移 | 0.1 | 0.2 | 效果不明显 |
| 高光调整 | 0.2 | 0.3 | 效果偏弱 |
| 阴影调整 | 0.2 | 0.3 | 效果偏弱 |
| 模糊混合 | 0.5 | 0.7 | 效果偏弱 |
